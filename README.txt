The following is a description of how to run the project.

SERVER:
Requirements: .NET runtime and SDK
Install from microsoft

build using 'dotnet build'
run using 'dotnet run'


Client:

Requirements:
python, flask package, flask_socketio package
Install using pip

Run command line client:

python client.python

Run GUI client:

python flask_app.py <port number>
Note: Do not use port 8080




Kill server and client from command line using Ctrl + C if an unhandled error occurs
